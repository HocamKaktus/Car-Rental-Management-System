# Car-Rental-Management-System
This is a project I did for my school quiz. It's a C based Car Rental Management System on system logic, data structures and transaction handling.

## What did I use?
- Language: C
- Logic: Data Structures (Struct) and Memory Management (Pointers).

## How does it work?
- You need a C compiler (like gcc or clang) to run this program.

## What does it do?
- Manages and organizes customer records.
- Adds and tracks cars for customers to rent.
- Schedules and handles car rentals.
